Each PNG here is a single 16x16 (player is 12x15) cell cut from those atlases:
  floor.png        <- tiles_sewers.png  cell (col 0,  row 0)
  wall.png         <- tiles_sewers.png  cell (col 0,  row 5)
  door.png         <- tiles_sewers.png  cell (col 8,  row 3)
  stairs_down.png  <- tiles_sewers.png  cell (col 0,  row 4)
  stairs_up.png    <- tiles_sewers.png  cell (col 12, row 3)
  potion.png       <- items.png         cell (col 0,  row 22)
  player.png       <- warrior.png       frame 0 (idle)
  goblin.png       <- gnoll.png          frame 0 (idle)   [gnoll stands in for "goblin"]
  skeleton.png     <- skeleton.png       frame 0 (idle)
